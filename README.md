## 手冊架構

依計畫生命週期重新編排為五大篇＋概述＋附錄：

- **概述** — 手冊使用說明、計畫總覽（五大補助類別比較）
- **第一篇 受理申請** — 申請流程、資格與文件審查、補件通知、公文併案／勾稽／改審議
- **第二篇 計畫審查** — 審查流程、初審意見製作、分組審查
- **第三篇 計畫審議** — 審議大會作業流程、辦理原則、核定通知（含過案/未過案公文範本）
- **第四篇 計畫管考** — 獎勵補貼請款、創新補助第一期款、實地查訪、計畫變更、結案與第二期款
- **第五篇 其他作業** — 諮詢輔導、廣宣、稽核、線上申請、合庫申貸
- **附錄** — 關鍵時限對照表、金額/比例對照、聯絡窗口、系統網址、新人入門檢核清單

## 功能特色

### 學習輔助
- 左側固定導覽列，可快速跳轉章節
- 橫向流程圖（5步驟卡片）與縱向時間軸並用
- 重要時限以 `badge` 標示（如「會前14天」「30日內」）
- 注意事項分四色：資訊（藍）／提醒（黃）／必做（紅）／完成（綠）
- 公文 / Email 範本以等寬字框呈現，可直接套用
- 附錄含「第一週／第一個月／三個月內」分階段檢核清單

### RWD 響應式設計
- **桌機** (>1024px)：280px 固定側欄，雙欄佈局
- **平板** (768~1024px)：240px 側欄，字級與內距縮小
- **手機** (≤768px)：側欄收合為漢堡選單（☰），流程圖自動垂直堆疊
- **超小螢幕** (≤380px)：進一步壓縮，stats 改單欄
- 表格自動橫向滾動，支援觸控
- 漢堡選單支援 ESC 關閉、點背景遮罩關閉

### 列印優化
- `@media print` 自動隱藏側欄、漢堡按鈕與遮罩
- 卡片、表格避免被切頁

### 直接瀏覽
雙擊 `115年度臺北市獎勵補助計畫_新人教育訓練手冊.html` 即可在瀏覽器中開啟，無需安裝任何套件。

1. 進入 repo → **Settings** → **Pages**
2. **Source** 選擇 `main` branch / `root`

## 維護指引

### 內容更新
直接編輯 HTML 檔內的對應 `<section>` 區塊。各章節 id 命名規則：

- `#ch1` ~ `#ch9`：主章節錨點
- `#ch1-flow`、`#ch1-check`：子章節錨點
- `#appendix`：附錄

### 樣式調整
所有 CSS 集中於 HTML `<head>` 內的 `<style>` 區塊。主要色票：

| 用途 | 色碼 |
|---|---|
| 主色（深藍） | `#1e3a5f` |
| 輔色（中藍） | `#2c5283` |
| 強調色（黃） | `#ffd54f` |
| 背景灰 | `#f4f6f9` |

### 新增章節
1. 在 `<aside>` 導覽列加入 `<a href="#new-id">` 連結
2. 在 `<main>` 內新增 `<section id="new-id">` 區塊
3. 使用既有的 `.card`、`.vflow`、`.flow`、`.alert` 等樣式類別
