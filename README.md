# 臺北市產業發展獎勵補助計畫 ─ 新人教育訓練資料庫

一套單頁 HTML 教育訓練資料庫，包含 **入口分類頁** + **兩本完整手冊**，協助臺北市產業獎勵補助專案辦公室新進同仁快速上手作業 SOP 並回應企業諮詢。

訪客一進入即可選擇要查閱的手冊類型，避免新人混淆：

## 手冊一：新人教育訓練手冊（內部 SOP）

整理自《115年度臺北市獎勵補助計畫作業手冊 SOP_final11504271810.docx》，依計畫生命週期重新編排為 9 大章節。

### 章節架構

| 章節 | SOP 頁碼 | 內容 |
|---|---|---|
| 1. 受理申請作業流程 | P.5 | 申請流程、資格審查、補件、改審議 |
| 2. 計畫審查作業 | P.12 | 初審意見、分組審查 |
| 3. 計畫審議 SOP | P.39 | 審議大會、核定通知、公文範本 |
| 4. 計畫管考 SOP | P.54 | 請款、實地查訪、計畫變更、結案 |
| 5. 計畫諮詢服務輔導 | P.133 | 諮詢申請、紀錄表 |
| 6. 計畫廣宣 | P.142 | 活動辦理、記者會、新聞稿 |
| 7. 稽核 SOP | P.155 | 稽核流程、確認表 |
| 8. 線上申請作業 SOP | P.158 | 申請人端／辦公室端操作 |
| 9. 其他公文 SOP | P.185 | 合庫申貸 |

### SOP 頁碼徽章

每個章節、小節與表單右側都標示對應的 SOP 原始頁碼，方便對照紙本：

- 黃色徽章「📖 SOP P.XX」── 章節頁碼（共 35 個）
- 藍色徽章「📋 表單 P.XX」── 表單／範本頁碼（共 21 個）

例如：
- 「2.2 初審意見製作 📖 SOP P.15-21」
- 「申請人開會通知單 📋 表單 P.32」
- 「請款應備文件 📋 表單 P.73-78」

## 手冊二：企業申請輔導手冊（對外應對）

整理自[官方網站](https://www.industry-incentive.taipei/)，協助同仁回應企業諮詢，並引導企業到正確頁面下載表單。

### 章節架構

- **快速開始** ── 8 個常用對外連結 + 計畫總覽（六大補助比較表）
- **六大補助詳解** ── 每項補助卡片化（資格／額度／範疇／文件 + 官網雙連結）
- **申請流程** ── 6 步驟流程圖 + 線上申請操作教學
- **表單與 SOP** ── 申請階段（7 連結）／執行階段（24 連結）／其他資源
- **應對指引** ── 17 題 FAQ 標準應答 + 8 個來電情境腳本 + 8 個情境處理
- **資源資訊** ── 完整聯絡方式 + 8 個生態夥伴連結 + 新人能力檢核

### 含 70+ 官方連結

所有表單、申請須知、SOP 文件都已嵌入官方原始 URL，同事接電話時可直接複製連結傳給企業，包括：

- 各補助申請書／計畫書／會計準則下載頁
- 第一期款、實地查訪、尾款各補助類別表單頁
- 「因應美國關稅，加碼 20% 獎勵補助」特別專案
- 審議委員會名單、歷年核定清冊
- 生態夥伴：創業臺北、臺北市投資服務辦公室、TOPS 台北好購等

## 共通功能

### RWD 響應式設計

四段斷點：桌機（>1024px）／平板（768~1024px）／手機（≤768px）／超小螢幕（≤380px）

- 桌機：280px 固定側欄 + 主要內容雙欄佈局
- 手機：側欄收合為漢堡選單（☰），流程圖自動垂直堆疊
- 表格自動橫向滾動，支援觸控

### 即時搜尋

兩本手冊都內建搜尋 bar，固定於頁面頂部：

- 輸入即時過濾（150ms debounce）
- 結果顯示章節麵包屑 + 關鍵字反白
- 點選跳轉 + 黃色閃爍提示
- 快捷鍵：`Ctrl/⌘+K` 喚出、`↑↓` 切換、`Enter` 跳轉、`Esc` 關閉

### 列印優化

`@media print` 自動隱藏側欄、漢堡按鈕、搜尋 bar、遮罩；卡片與表格避免被切頁。

## 使用方式

### 直接瀏覽

雙擊 `index.html` 即可在瀏覽器中開啟，選擇要查閱的手冊。無需安裝任何套件。

### GitHub Pages 部署

若要部署為線上手冊：

1. 進入 repo → **Settings** → **Pages**
2. **Source** 選擇 `main` branch / `root`
3. 儲存後即可透過 `https://<username>.github.io/<repo-name>/` 訪問入口頁

由於 `index.html` 為預設首頁，訪問根網址即顯示分類選單。

### 本機預覽

```bash
python3 -m http.server 8000
# 瀏覽器開啟 http://localhost:8000
```

## 維護指引

### 內容更新

兩本手冊各自獨立，直接編輯對應 HTML 檔內的 `<section>` 區塊。

### SOP 頁碼更新

若 SOP 文件改版，頁碼徽章需同步更新。HTML 中搜尋以下 CSS class：

- `sop-page sop-page-large` ── 章節頁碼徽章
